O agregador de links pastando oferece este plugin para facilitar a vida do webmaster. Ao enviar o post, o blog j&aacute; notifica o pastando do link.


== Description ==


O agregador de links funciona de maneira pr&aacute;tica e r&aacute;pida. Ao ativar o plugin, voc&ecirc; vai poder enviar atrav&eacute;s da p&aacute;gina de publica&ccedil;&atilde;o de novos posts,
seus links diretamente para o pastando.com.br. Que ir&aacute; divulgar seus links aumentar suas visitas de maneira bem legais.

O que voc&ecirc; ganha:

* Links sem a tag nofollow
* Aumento de Page Rank (Todos os links s&atilde;o diretos, n&atilde;o possuem a tag nofollow)
* Aumento de visitas
* conhecimento de outros webmasters, que sempre acessam o site para ver se o link foi aceito.

Algumas regras:

* Os links sem a miniatura n&atilde;o ser&atilde;o aceitos
* Voc&ecirc; s&oacute; pode enviar 3 links di&aacute;rios de cada blog
* Imagens de sexo expl&iacute;cito n&atilde;o ser&atilde;o aceitos (possu&iacute;mos categoria adulta, mas seja moderado)
* Envite t&iacute;tulos muitos longos. Se tiver muitos caracteres o link n&atilde;o ser&aacute; aceito.


Voc&ecirc; pode enviar seu link de posts antigos, se ainda forem atrativos e ainda n&atilde;o tiverem sido publicados no pastando.com.br.
Basta voc&ecirc; acessar a edi&ccedil;&atilde;o do post e preencher as informa&ccedil;&otilde;es, aceitando postar no pastando. Simples.

COMO VOC&ecirc; SABE QUE O LINK J&aacute; FOI ENVIADO?

Ao preencher os dados e publicar/atualizar o post, voc&ecirc; ver&aacute; uma mensagem assim: "Este link j&aacute; existe no pastando.com.br".

Isso significa que o envio ocorreu com sucesso. Pode aguardar que se seu post n&atilde;o tiver nada de errado, ele ser&aacute; aceito muito breve. 

LINKS DI&aacute;RIOS

Os link s&atilde;o aceitos di&aacute;riamente, isso significa que seu post ser&aacute; aceito no pr&oacute;prio dia ou no dia seguinte na maioria das vezes.

1. A partir da&iacute;, &eacute; s&oacute; voc&ecirc; ir na p&aacute;gina de publica&ccedil;&atilde;o/edi&ccedil;&atilde;o de posts e visualizar o bloco do pastando.com.br

== Frequently Asked Questions ==

= Acabei de postar o link e d&aacute; a mensagem "Este link j&aacute; existe no pastando.com.br", deu erro? =

N&atilde;o, o que aconteceu foi que seu link foi enviado com sucesso, e quando o navegador recarregou a p&aacute;gina o sistema fez uma outra valida&ccedil;&atilde;o
para saber se o link foi postado, a&iacute; ele diz que j&aacute; foi. Sempre que aparecer esta mensagem n&atilde;o se preocupe, seu link j&aacute; foi enviado para o pastando.com.br

= Quantos links posso enviar diariamente? =

Voc&ecirc; pode enviar 3 links. N&atilde;o se preocupe em ficar contando. Nosso sistema o avisar&aacute; e voc&ecirc; nem poder&aacute; enviar mais de 3 no mesmo dia.
